## Pincode Accounting
Two-factor authentication accounting service.

## Setup
1. Setup and start main service: https://github.com/IlyaDonskikh/pincode.
2. Clone current repository.
3. Start rails server.
4. Open site and follow the instructionsy.

## Bonuces
